
# HR Analytics – Employee Attrition Analysis

This project analyzes employee attrition data using Power BI and Excel.

## Overview
The dashboard focuses on:
- Employee attrition rate
- HR KPIs
- Department and role-based analysis
- Workforce trends

## Tools
- Power BI
- DAX
- Microsoft Excel
- Data Modeling
- HR KPIs
